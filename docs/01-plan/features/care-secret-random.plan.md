# 돌봄/비밀진화/랜덤이벤트 통합 기획서

> **Summary**: 다마고치 핵심 흥행 요인 접목 - 돌봄 시스템, 비밀 진화, 비밀 상점/랜덤 이벤트
> **Date**: 2026-03-18

---

## 기능 A: 돌봄 시스템 + 일일 루틴

### 3대 게이지
- **배고픔** (hunger): 시간 경과로 감소, "밥주기"로 회복
- **청결** (clean): 미션 안 하면 감소, "씻기기"로 회복
- **행복** (fun): 미니게임/미션으로 회복, "놀아주기"로 회복

### 게이지 규칙
- 최대 100, 로그인 시 경과 시간 기반으로 감소 계산
- 시간당 배고픔 -2, 청결 -1, 행복 -1.5
- 0 이하로 3일 유지 시 "아픔" 상태 → 경험치 획득 50% 감소
- 돌봄 행위: 1일 각 1회, 게이지 +40 회복

### 일일 루틴
- 밥주기 + 미션 1개 + 미니게임 1회 = "완벽한 하루" 보너스 (+20 EXP)

## 기능 B: 비밀 진화

### 4가지 비밀 형태 (기존 최종 진화와 별도)
- **수학왕**: 미니게임 만점 20회 + INT 15↑
- **불사조**: 아팠다 회복 3회 + STA 10↑
- **무지개**: 모든 색상 + 모든 눈 스타일 보유
- **전설**: Lv.20 + 모든 능력치 10↑

### 힌트 시스템
- 도감에서 "???" 형태로 표시, 조건 근접 시 힌트 메시지
- 비밀 진화는 기존 최종 진화 위에 "덮어쓰기" (더 상위)

---

## 기능 C: 비밀 상점 & 랜덤 이벤트

### 일일 할인 상점
- 매일 1개 아이템 50% 할인 (날짜 기반 시드로 결정)

### 행운의 룰렛
- 연속출석 7일마다 1회 → 포인트/아이템/간식상자 랜덤

### 깜짝 이벤트
- 미션 완료 시 10% 확률로 "보물 발견!" → 추가 포인트
