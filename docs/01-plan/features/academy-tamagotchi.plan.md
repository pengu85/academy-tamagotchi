# 학원 다마고치 프로그램 기획서

> **Summary**: 학원 미션 수행으로 다마고치를 육성하고, 포인트로 커스터마이징하며, 학원 운영과 연계하는 게이미피케이션 시스템
>
> **Project**: 학원다마고치프로그램
> **Author**: postmath21
> **Date**: 2026-03-18
> **Status**: Draft

---

## 1. 개요

### 1.1 목적

학원 학생들의 미션(과제, 출석, 시험 등) 수행 동기를 높이기 위해, 미션 완료 시 다마고치를 육성할 수 있는 게이미피케이션 웹 프로그램을 개발한다.

### 1.2 배경

- 학생들이 학원 미션을 수동적으로 수행하는 문제
- 게이미피케이션을 통해 자발적 참여와 성취감 유도
- 다마고치 육성이라는 친숙한 게임 요소로 학생 몰입도 향상
- 학원 운영(간식상자, 경연대회 등)과 연결하여 오프라인 보상 체계 구축

### 1.3 관련 문서

- 없음 (신규 프로젝트)

## 6. 아키텍처

### 6.1 프로젝트 레벨

| 레벨 | 특성 | 대상 | 선택 |
|------|------|------|:----:|
| **Starter** | 간단한 구조 (components/, lib/, types/) | 정적 사이트, 포트폴리오 | **V** |
| Dynamic | 기능 기반 모듈, BaaS 연동 | 백엔드 포함 웹앱 | |
| Enterprise | 계층 분리, DI, 마이크로서비스 | 대규모 시스템 | |

### 6.2 주요 기술 결정

| 결정 | 선택지 | 선택 | 이유 |
|------|--------|------|------|
| 기본 기술 | HTML/CSS/JS | HTML/CSS/JS | Starter 레벨, 별도 프레임워크 불필요 |
| 스타일링 | Vanilla CSS / Tailwind | Vanilla CSS | 의존성 최소화 |
| 데이터 저장 | localStorage | localStorage | 서버 없이 클라이언트에서 데이터 유지 |
| 캐릭터 렌더링 | 이미지 / CSS / SVG / Canvas | CSS + SVG | 가볍고 커스터마이징 용이 |
| 빌드 도구 | 없음 (순수 HTML) | 없음 | 단순한 구조 유지 |

### 6.3 폴더 구조 (예상)

```
학원다마고치프로그램/
```

## 7. 핵심 게임 시스템 설계 (개요)

### 7.1 레벨/경험치 시스템

| 레벨 | 필요 경험치 | 누적 경험치 | 보상 |
|------|------------|------------|------|
| 1 | 0 | 0 | 다마고치 생성 |
| 2 | 100 | 100 | 포인트 5p |
| 3 | 150 | 250 | 포인트 5p |
| 4 | 200 | 450 | 포인트 10p + 색상 해금 |
| 5 | 300 | 750 | 포인트 10p + 간식상자 기회 |
| ... | 점진적 증가 | ... | 레벨별 보상 |

### 7.2 미션 유형 및 경험치

| 미션 유형 | 경험치 | 예시 |
|-----------|--------|------|
| 출석 | 10 | 오늘 학원에 왔어요 |
| 과제 완료 | 30 | 숙제를 제출했어요 |
| 시험 성적 | 20~50 | 시험 점수에 따라 차등 |
| 특별 미션 | 50~100 | 선생님이 부여하는 특별 과제 |

### 7.3 능력치 특화 시스템

| 능력치 | 효과 | 관련 보상 |
|--------|------|-----------|
| 힘 (STR) | 경연대회 '파워' 부문 | 특별 이펙트 |
| 지능 (INT) | 퀴즈 보너스 경험치 | 추가 경험치 획득 |
| 매력 (CHA) | 경연대회 '인기' 부문 | 투표 보너스 |
| 체력 (STA) | 연속 출석 보너스 | 간식상자 확률 증가 |

### 7.4 아이템/외형 커스터마이징

| 카테고리 | 예시 | 포인트 비용 |
|----------|------|------------|
| 색상 | 몸 색상 변경 | 3p |
| 모자 | 왕관, 야구모자, 머리띠 | 5p |
| 악세서리 | 안경, 목걸이, 날개 | 5~10p |
| 배경 | 우주, 바다, 숲 | 8p |
| 특수 효과 | 반짝이, 후광, 불꽃 | 15p |
