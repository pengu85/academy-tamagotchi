## 1. 개요

### 1.1 목적

다마고치가 레벨에 따라 **단계적으로 외형이 변화(진화)**하고, 학생이 투자한 능력치에 따라 **최종 진화 형태가 달라지는** 시스템을 추가한다. 이를 통해 학생들에게 장기 목표를 제공하고, "나만의 유일한 다마고치"라는 소유감을 강화한다.

### 1.2 배경

- 현재 시스템은 레벨이 올라도 다마고치 외형이 동일 (아이템 제외)
- 학생들이 레벨업 이후 "다음 목표"가 부족하여 흥미 저하 가능
- 진화 분기를 통해 능력치 배분에 전략적 의미를 부여
- "내 다마고치는 어떤 형태로 진화할까?"라는 기대감으로 몰입도 향상

### 1.3 관련 문서

- 기존 기획서: [academy-tamagotchi.plan.md](./academy-tamagotchi.plan.md)
- 기존 설계서: [academy-tamagotchi.design.md](../../02-design/features/academy-tamagotchi.design.md)

## 2. 범위

### 2.1 포함 범위 (In Scope)

- [ ] 5단계 진화 체계 (알 → 아기 → 어린이 → 청소년 → 최종)
- [ ] 레벨 기반 자동 진화 트리거
- [ ] 능력치 기반 최종 진화 4분기 (용사/학자/스타/수호자)
- [ ] 진화 단계별 SVG 캐릭터 외형 변화
- [ ] 진화 시 축하 연출 (모달 + 애니메이션)
- [ ] 홈 화면에 현재 진화 단계 표시
- [ ] 진화 도감 (어떤 진화 형태가 있는지 미리보기)

### 2.2 제외 범위 (Out of Scope)

- 진화 되돌리기 (한번 진화하면 유지)
- 진화에 따른 추가 스킬/능력 (외형 변화만)
- 융합 진화 (능력치 2개 이상 조합)

## 4. 진화 체계

### 4.1 진화 단계 (5단계)

| 단계 | 이름 | 진화 레벨 | 외형 특징 | 크기 비율 |
|:----:|------|:---------:|-----------|:---------:|
| 1 | 알 (Egg) | Lv.1 | 둥근 알, 금이 살짝 | 0.6x |
| 2 | 아기 (Baby) | Lv.3 | 작고 둥근 몸, 짧은 팔다리 | 0.8x |
| 3 | 어린이 (Child) | Lv.6 | **현재 기본 형태** (호환) | 1.0x |
| 4 | 청소년 (Teen) | Lv.10 | 몸이 약간 길어지고, 귀/뿔 등 특징 생김 | 1.1x |
| 5 | 최종 (Final) | Lv.15 | **능력치 분기** - 4가지 형태 | 1.2x |

> **핵심**: Lv.6 "어린이" 단계가 현재 기본 캐릭터 형태와 동일하므로, 기존 데이터와 자연스럽게 호환된다.

### 4.2 최종 진화 분기 (Lv.15)

가장 높은 능력치에 따라 4가지 형태로 분기:

| 최고 능력치 | 진화 형태 | 외형 특징 | 컨셉 |
|:-----------:|-----------|-----------|------|
| STR (힘) | 용사 (Warrior) | 각진 몸, 작은 뿔, 강인한 눈 | 힘세고 용감한 전사 |
| INT (지능) | 학자 (Scholar) | 큰 머리, 반짝이는 눈, 책/지팡이 | 지혜로운 마법사/학자 |
| CHA (매력) | 스타 (Star) | 날씬한 몸, 반짝이, 왕관 느낌 | 인기 많은 아이돌 |
| STA (체력) | 수호자 (Guardian) | 큰 몸, 방패 느낌, 든든한 체형 | 든든한 수호자 |

> **동률 시**: STR > INT > CHA > STA 우선순위로 결정

### 4.3 진화 흐름

```
Lv.1 [알] → Lv.3 [아기] → Lv.6 [어린이] → Lv.10 [청소년]
                                                    │
                                              Lv.15 최종 진화
                                         ┌─────┬─────┬─────┐
                                         │     │     │     │
                                     STR최고 INT최고 CHA최고 STA최고
                                      용사   학자   스타   수호자
```

## 5. 데이터 변경사항

### 5.1 Student 데이터 추가 필드

```javascript
// 기존 tamagotchi 객체에 추가
tamagotchi: {
  // ... 기존 필드 유지
  evolution: {
    stage: 3,              // 현재 진화 단계 (1~5)
    stageName: "어린이",    // 단계 이름
    finalForm: null,       // 최종 진화 형태 (null | "warrior" | "scholar" | "star" | "guardian")
  }
}
```

### 5.2 하위 호환성

기존 학생 데이터에 `evolution` 필드가 없는 경우:
- 레벨에 따라 자동 계산하여 적절한 단계 부여
- Lv.1~2 → stage 1(알), Lv.3~5 → stage 2(아기), Lv.6~9 → stage 3(어린이), Lv.10~14 → stage 4(청소년), Lv.15+ → stage 5(최종)

## 7. 성공 기준

### 7.1 완료 정의

- [ ] 5단계 진화가 레벨에 따라 자동 적용
- [ ] 4가지 최종 진화 형태가 능력치에 따라 올바르게 분기
- [ ] 기존 학생 데이터 로그인 시 정상 동작 (하위 호환)
- [ ] 기존 아이템이 모든 진화 단계에서 정상 장착
- [ ] 진화 축하 모달 정상 표시

### 7.2 품질 기준

- [ ] 모든 진화 형태의 SVG가 정상 렌더링
- [ ] 진화 시 깜빡임/레이아웃 깨짐 없음
- [ ] 브라우저 새로고침 후 진화 상태 유지
